# 🌸 RiAtlas Task Manager

A modern and elegant Flutter task management application inspired by Moroccan aesthetics. RiAtlas Task helps users stay organized with task categories, priorities, due dates, productivity tracking, and a beautiful user interface.

---

## ✨ Features

- ✅ Create, edit, and delete tasks
- 📂 Organize tasks with custom categories
- 🔥 Priority management (High, Medium, Low)
- 📅 Due dates and calendar view
- 📊 Productivity statistics
- 👤 User profile customization
- 🎯 Daily productivity insights
- 💾 Offline local storage using Hive
- 🎨 Beautiful pink modern UI

---

## 🛠️ Built With

- Flutter
- Dart
- Hive Database
- Material Design 3

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/GHITAOUARZI2004/riatlas-task-manager.git
```

### Navigate to the project

```bash
cd riatlas-task-manager
```

### Install dependencies

```bash
flutter pub get
```

### Run the app

```bash
flutter run
```
